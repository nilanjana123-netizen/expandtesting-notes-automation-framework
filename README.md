ExpandTesting Notes Hybrid Automation Framework
Project Overview

This project is a Hybrid UI + API Automation Framework developed for the ExpandTesting Notes Application using Selenium WebDriver, Java, TestNG, RestAssured, Maven, Allure, and JMeter. The framework validates note management functionalities such as login, note creation, editing, deletion, and synchronization between UI and API layers.

The framework follows enterprise automation practices including Page Object Model (POM), reusable utility methods, ThreadLocal driver management, explicit waits, reporting integration, and modular framework architecture.

Features
UI Automation using Selenium WebDriver
API Automation using RestAssured
Hybrid UI + API Validation
Page Object Model (POM) Design
ThreadLocal WebDriver Management
Explicit Wait and JS Executor Utilities
Excel Data Parameterization
Allure Reporting Integration
Screenshot Capture on Failure
JMeter Performance Validation
Modular and Reusable Framework Components
